# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-30

### Added
- Initial release of Airview flight tracking application
- Real-time flight tracking using FlightRadar24 API
- Responsive web interface with kiosk mode support
- Admin panel for configuration management
- Docker containerization for easy deployment
- Aircraft type enhancement with full descriptive names
- Airport code enhancement with city names
- Configuration persistence via JSON files
